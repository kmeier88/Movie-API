Movie API

This is an API that will return data about movies and allow users to create accounts and keep track of favorite movies.

Description of Endpoint Endpoint URL Method Type Query Parameters Format of Request Format of Response
Return a list of ALL movies to the user /movies GET None None A JSON object holding a list of all the movies
Return data about a single movie by title to the user /movies/[Title] GET [Title] None A JSON object holding data about a single movie, containing a description, genre, director, image URL. Example:
{
title: "Legally Blonde",
description: "This movie isn't very good",
genre: "comedy (kinda)",
director: "Who cares",
image URL: "image"
{
Return data about a genre by title of movie /movies/[Title]/[Genre] GET [Genre], [Title] None A JSON object holding data about the genre of the movie by title. Example:
{
title: "Legally blonde",
genre: "comedy"
}
Return data about a director (bio, birth year, death year) by name /movies/director/[Director] GET [Director], [Bio], [Birth year], [Death year] None A JSON object holding data about a director. Example:
{
director: "Julio Martinez",
bio: "Julio is a wicked good director",
birth year: "1969",
death year: "N/A",
}
Allow new users to register /users POST [Username], [DOB] A JSON object holding data about the new user
{
name: "Jerry Seinfeld",
DOB: "4/20/69"
} 		A JSON object holding data about the new user that was added, including a username. Example:

{
username: "JS42069",
name: "Jerry Seinfeld",
DOB: "4/20/69"
}
Allow users to update their user info (username) /users/[Username] PUT [Username] None Text showing that the username has been updated
Allow users to add a movie to ther list of favorites /users/[Favorites] PUT [Favorites] None Text saying that a movie has been added
Allow users to remove a movie from their list of favorites /users/[Favorites] DELETE [Favorites] None Text showing that a movie has been removed from their list of favorites
Allow existing users to deregister /users DELETE Query Paramters IDK None Text showing that the user has been deregistered
